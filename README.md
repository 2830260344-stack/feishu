# AI 视频运营助手 🤖
基于飞书机器人 + 小龙虾 AI 的视频运营助手，可在飞书内直接调用 AI 能力，辅助视频创作与运营。

## 功能特性
- 🤖 飞书机器人：在飞书内直接与 AI 对话
- 🎬 视频运营：爆款分析、文案创作、运营计划制定
- ☁️ 云部署：支持 Render 一键部署，公网可访问
- 🔒 环境变量：敏感信息通过环境变量配置，安全可靠

## 快速开始

### 1. 环境准备
- Node.js 16+
- 飞书开放平台应用（获取 `APP_ID` 和 `APP_SECRET`）
- 小龙虾 AI API 地址与密钥
